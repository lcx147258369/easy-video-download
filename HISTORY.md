# History

## 2026-06-06

### 下载状态与资源操作优化
- 强化下载成功状态展示，让已完成资源更容易识别。
- 移除了资源上的“重新抓取”按钮。
- 新增“查看下载内容”功能，下载完成后可直接预览本地内容。

### 设置项收敛
- 屏蔽了 UI 中的“无头浏览器模式”入口，改为默认内部使用。
- 自动勾选 `autoDownload`，并隐藏对应入口。
- 将下载路径和浏览器 Profile 路径从手动输入改成系统目录选择。

### 页面任务主流程重构
- 输入方式统一为“输入 URL”。
- 主流程调整为：`输入 URL -> 页面任务 -> 抓取资源 -> 生成下载任务`。
- 主页面改为以页面任务列表和详情联动为主，不再以旧的全局资源队列为主入口。

## 2026-06-06 / 2026-06-07

### 页面任务删除能力
- 新增页面任务删除功能。
- 支持页面任务多选。
- 支持“全选页面任务”。
- 支持“删除已选”批量删除，前端通过现有单删接口逐个删除。
- 删除后会同步清理前端选中状态，避免右侧详情停留在已删除任务上。

### 稳定性修复
- 修复了 `task-detail.tsx` 中不存在的 `DownloadStatus` 类型引用，改为基于 `TaskDetailResponse` 推导资源下载状态类型。
- 更新了与新首页标题和批量删除逻辑相关的前端测试。
